# Advertising Solutions Email Metrics Dashboard

Static HTML dashboard for Crain Advertising Solutions email campaign performance.

## Scope

- Custom email campaigns only
- Modern Healthcare, Ad Age, Pensions & Investments, GenomeWeb, Crain's Chicago, and Crain's Detroit
- Sailthru campaign performance data from January 1 through May 23, 2026
- Modern Healthcare conversion context from the Ad Solutions master tracker

## Files

- `index.html` - standalone dashboard

Open `index.html` in a browser to view the dashboard. Charts use Chart.js from a CDN.
